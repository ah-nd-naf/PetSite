# 🐾 PetSite – Adopt Your Perfect Pet

## 🌐 Live Demo
PetSite is live!  
👉 [View the site here](https://pet-site-pi.vercel.app/)


PetSite is a responsive, modern web application designed to connect loving families with pets in need of a home. Built with **HTML, Tailwind CSS, and Swiper.js**, it offers a clean, engaging interface that feels like a real social platform.

---

## ✨ Features
- **Hero Section**: Eye‑catching banner with call‑to‑action.
- **Pet Gallery**: Card‑based layout showcasing pets with hover effects.
- **Adoption Process**: Clear 3‑step guide to adopting a pet.
- **Live Adoption Form**: Integrated with Formspree for real submissions.
- **Happy Families Slider**: Auto‑sliding testimonials powered by Swiper.js.
- **Responsive Design**: Optimized for desktop and mobile.

---

## 🚀 Tech Stack
- **Frontend**: HTML, Tailwind CSS
- **Interactivity**: Swiper.js (carousel), vanilla JS (menu toggle)
- **Form Handling**: Formspree integration
